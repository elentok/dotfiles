elentok's dotfiles
=======================

To install run:

```bash
  $ curl -L https://github.org/elentok/dotfiles/raw/master/online_install.sh | bash
```

Customization
--------------

* Add your git information to ~/.dotlocal/gitconfig:

```gitconfig
[user]
  name = Your Name
  email = you@gmail.com
[github]
  user = your-github-user
```

* Add your custom aliases to ~/.dotlocal/zshrc.after
