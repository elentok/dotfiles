# Workflow

Perform retrospective review of this session:

1. What went well?
2. What didn't go well?
3. What would you have done differently?
4. Update .ai/index.md and the rest of the docs in .ai/ to make this smoother next time
5. Write the retro summary to `docs/retro/` and use sequential numbering: `00001-slug.md` (where
   `slug` is the short name for the session)
6. Show the summary to the user
