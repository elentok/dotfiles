Vim
=====
* z= - suggest correct spelling
* ]q [q - cnext/cprevious
* ]n [n - prev/next conflict markers


Other Apps
==========

Pixelmator
----------
v - move
z - zoom
m - rectangular select (marquee)

iMessages
----------

* Cmd + [,] - previous/next chat
* Cmd + F - go to search bar
* Cmd + F2 - bring to focus

Gmail
----------

* # - delete
* e - archive
* x - select
* v - move to
* gi - go to inbox
